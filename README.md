# Spring and Ionic Fullstack Course
Spring + Ionic Project made during Udemy Course.

## API Side
- Spring Boot
- Spring MVC
- Spring Security (JWT)
- Spring JPA (Hibernate)
- Spring Mail (with ThymeLeaf)
- MySQL
- AWS S3
- ImgScalr (Image cropping, resizing)

## Mobile/Ionic Side
- Ionic 3
- Angular 5
- Typescript
- Camera, Storage, JWT, Lazy Loading
